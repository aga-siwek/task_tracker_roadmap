# TASK TRACKER

## Project Description

Project for manage task

## How to Set Up Project

create virtual env

`python -m venv .venv`

activate virtual env

```
# windows (cmd)
.\.venv\Scripts\activate.bat

# windows (PowerShell)
.\.venv\Scripts\activate.ps1

# macos, linux
source .venv/bin/activate
```

install project dependencies

`pip install .`
